# MR Flashing (Web PWA)

This is the MR Flashing web app for drawing, pricing, and ordering custom flashings.

### Features
- Drawing canvas → save templates
- Orders with pricing (base rate × gauge factor)
- Supplier address book + default supplier
- Email order + Export PDF
- Works offline (PWA)

### Deploy on iPad (Vercel)
1. Download and unzip this folder.
2. In GitHub → your repo → **Add file → Upload files** → open this folder → **Select All** → **Commit changes**.
3. On Vercel → **Add New → Project** → import your repo → **Deploy**.
4. Open your `.vercel.app` URL → Safari → **Share → Add to Home Screen**.
